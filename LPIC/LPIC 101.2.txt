Запуск системы

1 BIOS / UEFI  # Запуск системы
2 MBR / GPT    # Первая стадия загрузки (partition table)
3 LILO / GRUB  # Вторая стадия загрузки
4 Linux Kernel # Запуск ядра ОС
5 Init         # Пользовательская среда

Загрузчики

NTLDR     # Windows NT
LILO      # Linux и BSD - системы
GRUB      # загрузчик ОС от проекта GNU
BootX     # MaxOS
Chameleon # гибкий кроссплатформенный загрузчик

При старте -> (shift) / (boot + ESC) -> меню загрузки -> выбрать пункт и нажать i для редактирования кастомной текущей загрузки

dmesg # показать ВСЕ произошедшие сообщения ядра
cat /var/log/dmesg # показать произошедшие сообщения ядра только во время загрузки
